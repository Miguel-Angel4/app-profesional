Manual de Usuario

Guía completa para el uso de la aplicación App Profesional

Volver a la App

1 Descripción de pantallas

La aplicación consta de las siguientes pantallas principales, accesibles desde el menú lateral:

Inicio

Panel de control principal con resumen de actividad y acceso a tareas relevantes.

Listado

Página central de gestión, donde se muestran listados con acciones interactivas.

Informe

Módulo de estadísticas y resúmenes para analizar los datos introducidos.

Ayuda

Sección informativa que da acceso al manual y ayuda contextual del sistema.

2 Flujo de uso

El flujo general para interactuar con la aplicación es el siguiente:

1

Revisar Inicio

Ingresa al sistema para tener una visión rápida de tareas pendientes o recientes en la pantalla de Inicio.

2

Gestionar en Listado

Dirígete al Listado para interactuar (agregar, editar, borrar) con los registros usando el botón Acciones.

3

Analizar y Exportar

Ve a la vista de Informe para observar estadísticas agregadas o exportar el progreso de tus tareas.

3 Instalación Android

Para disfrutar de la experiencia móvil nativa, instala el archivo APK siguiendo estos pasos:

  1. Permitir orígenes desconocidos: Ve a Ajustes > Seguridad (o "Instalar aplicaciones desconocidas") en tu dispositivo Android y actívalo.
  2. Obtén el archivo app-profesional.apk desde el directorio de descargas o repositorio oficial de la aplicación.
  3. Utiliza un explorador de archivos para localizar el archivo descargado.
  4. Toca el archivo `.apk` y selecciona Instalar. Una vez finalizado, puedes abrir la app.

4 Configuración básica

Antes de comenzar a trabajar formalmente, asegúrate de configurar estos aspectos fundamentales:

  • Permisos y Almacenamiento: Si se te solicita, otorga acceso a almacenamiento (para informes) y notificaciones en tu dispositivo.
  • Sincronización Inicial: Verifica que los listados cargan correctamente; requerirá conexión a internet activa para obtener el catálogo base.
  • Modo de Ayuda Contextual: En caso de duda durante una pantalla específica, busca el icono ? presente en cada sección para resolver dudas sin salir de tu tarea.
